## Topics
- Neural network layers: input (data entry), hidden (processing), output (results)
- Network parameters: weights (connection strength), biases (offsets), values (computed outputs)
- Activation functions: ReLU (positive passthrough), Sigmoid (0-1 scaling), Tanh (-1 to 1 scaling)
- Loss functions: MSE (regression errors), Cross-entropy (classification errors), Custom (specialized needs)
- Training cycles: epochs (full data passes), early stopping (preventing overfitting), embeddings (categorical encoding)
